## 2025 Competition Repository

This repo contains 8229's final competition code for the 2025 FRC game Reefscape at Tidal Tumble. 

### Overview

- Command-Based Framework
- CTRE Swerve Template Base
- Pathplanner Autonomous
- TalonFX Controllers
- Megatag 1 Vision

### Subsystems

**Swerve** - Uses four Falcons for angle motors, and four Krakens for drive motors. Utilizes the [CTRE Swerve Project Generator](https://pro.docs.ctr-electronics.com/en/latest/docs/tuner/tuner-swerve/index.html) as a template.

**Arm** - Runs off of a Kraken motor and uses the internal encoder for position, used to articulate the manipulator 180+ degrees.

**Elevator** - Runs off of two Falcon motors and uses the internal encoder for position, used to elevate the manipulator.

**Intake** - Runs off of three Falcon motors, and is used to deploy the intake, grab the coral, and center the coral.

**Claw** - Runs off of one Kraken motor. It is used to hold and score game pieces.
